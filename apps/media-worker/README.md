# Mecorion Media Worker

Media Worker будет отвечать за фоновые задачи:

- чтение исходников из `data/temp` или будущего S3;
- анализ аудио и видео через FFmpeg;
- создание обработанных версий файлов;
- запись результата в `data/music`, `data/video`, `data/books`;
- обновление статусов задач в PostgreSQL.

Пока это только каркас. Логика появится после финализации API загрузки файлов.
